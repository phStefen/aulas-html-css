Elementos Block

São Elementos que ocupam toda a Largura da Tela. Ex:

h1

p

div

Elementos InLine:

São Elementos que sua a Largura depende do conteúdo, e eles ficam em linhas. Ex:

span span

Elementos Block com display Inline:

h1

h1

Elementos InLine com display Block:

span span

Elementos com display Inline-Block:

Fica um ao lado do outro enquanto cabe na mesma linha, se não couber ele cai pra linha de baixo
